Forms

Form Example for exercise

1. Accessing Forms

a. const form = document.forms[0]; // uses index notation to return the first form object
b. const form = document.forms.search; // uses the name attribute to identify a form
c. const form = document.forms['search']; // square bracket notation is recommended to avoid referencing any properties or methods in the form
d. const input = form.searchInput or const input = form['searchInput'] to access the input field named searchInput

2. a. form.submit() - submits the form automatically or can use:
button type='submit'>Submit
input type='submit' value='Submit'>
input type='image' src='button.png'>

b. form.reset() or button type='reset'>Reset // resets the form, but be careful in using it since it will wipe out all the data that has been entered.
c. focus event - is used to put the focus on an element, like when you put your cursor in the input field
d. change event - occurs when the user moves the focus away from the form element after changing it

3. submit event - occurs when the form is submitted, will send the content of the form to the server to be processed

4. Form Controls - Hero.html

input -fields, including text, passwords, check boxes, radio buttons, and file uploads
select - menus for drop-down lists of options
textarea - elements for longer text entry
button -elements for submitting and resetting forms

a. autofocus attribute give focus to this element when a page loads - it is the same as the focus() event
b. maxlength - limits the number of characters to be entered in the field
** There are still many attributes that can be used in a form to make it user-friendly

5. Input fields

a. input type='password' is used to enter passwords or secret information - the characters are concealed
b. type='checkbox' - used to select different options, can check more than one
c. type='checkbox' value='Flight' name='powers' checked -- checked attribute is being used to initially check the checkboxes
d. radio button - only one option can be selected
e. type='file' - use to upload files

6. disabled attribute - disable the submit button when there are errors on the form
